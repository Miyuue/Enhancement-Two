You will see the config files in the "Scene" folder.
Each file within will be an example file with explanations you can read in order to
craft your own Scene, CompositeMeshObject, Material, and Light files for use in the program.

I crafted two scenes for a demonstration of the reload feature.
Simply change the first line in Scene/scene_config.txt to the name of any scene in the "Scenes" folder.
The two scenes I crafted are called "example" and "smiley_face". 

After updating the scene name in that Scene/scene_config.txt, return to the program and press "R". 
You'll then see the scene update accordingly. Of course, this is the same with all the config files in the "Scene" folders.

Controls:

Camera Movement: W,A,S,D, E,Q

Camera Rotation: Move mouse

Camera Movement Speed: Adjusted by scrolling

Change camera view: P,O

Reload Scene: R
